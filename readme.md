# Carrinho da Shopee

Projeto desenvolvido no bootcamp da DIO para praticar lógica de programação com JavaScript. A aplicação simula um carrinho de compras inspirado na Shopee, permitindo adicionar produtos, visualizar itens, remover quantidades e finalizar uma compra pelo terminal.

## Funcionalidades

- Listar produtos disponíveis.
- Adicionar itens ao carrinho.
- Visualizar produtos escolhidos.
- Remover quantidades do carrinho.
- Calcular o total da compra.
- Simular formas de pagamento.

## Tecnologias

- JavaScript
- Node.js
- Readline

## Como executar

Clone o repositório e acesse a pasta do projeto:

```bash
git clone https://github.com/bebelle20092020j/carrinho-shoppe.git
cd carrinho-shoppe
```

Execute o arquivo principal:

```bash
node src/index.js
```

## Estrutura

```txt
src/
  services/
    cart.js
    item.js
  index.js
```

## Sobre o projeto

O projeto foi criado para representar as principais ações de um carrinho de compras. O arquivo `services/item.js` cria os produtos, o `services/cart.js` concentra as funções do carrinho e o `index.js` controla a interação com o usuário pelo terminal.
